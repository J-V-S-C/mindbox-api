# MindBox - Productivity application designed to help people organize their minds

## Features

### Roadmap

`Roadmap` -> `Category` -> `Tasks`

- Each category has an expiration date.
- If the deadline is reached and tasks aren't finished, the category is automatically flagged as `EXPIRED`.

### Daily

`Check-list` | `Learning Log`

- **Habit Tracker:** Recurring tasks that reset every day at 00:00.
- **"X Things I Learned Today":** A micro-journaling section to document daily technical growth and build a "Knowledge Portfolio."
- **Persistence:** Daily logs are stored to track long-term consistency and learning history.

### Books

`PDF Reader`

- **Library:** Store technical books with metadata and progress tracking.
- **Attachments:** Native support for uploading and reading PDFs directly in the browser.
